# 📊 Business Audit Pro

> Marketing va Moliyaviy Audit Web Ilovasi

**Business Audit Pro** — kompaniyangizning marketing va moliyaviy holatini 3 daqiqada chuqur tahlil qiladigan zamonaviy web ilova.

## ✨ Xususiyatlar

- 🏢 **Biznes diagnostikasi** — CRM, sotuv bo'limi, ijtimoiy tarmoq holati
- 💰 **Moliyaviy maqsad tahlili** — Funnel hisob-kitobi, byudjet optimizatsiyasi
- 📈 **Marketing audit** — Platforma tahlili, CPL benchmark, konversiya
- 📊 **Interaktiv dashboard** — Grafiklar, yo'qotishlar, roadmap
- 💡 **Termin izohlari** — Har bir texnik atama yonida tushuntirish
- 📱 **To'liq responsive** — Mobil, planshet va desktop da ishlaydi

## 🛠 Texnologiyalar

- **React 18** + **Vite**
- **Recharts** — ma'lumotlar vizualizatsiyasi
- **Vanilla CSS** — glassmorphism dark mode dizayn
- **JavaScript** — audit hisoblash mexanizmi

## 🚀 Lokal ishga tushirish

```bash
# Dependencies o'rnatish
npm install

# Development server
npm run dev

# Production build
npm run build
```

## 📐 Audit modullari

| Modul | Savollar | Nima tahlil qiladi |
|---|---|---|
| **Biznes holati** | 5 ta | CRM, sotuv bo'limi, ijtimoiy tarmoq |
| **Moliyaviy maqsad** | 4 ta | Daromad, chek, byudjet |
| **Marketing** | 2 ta | Platforma, konversiya |
| **Sotuv tizimi** | 2 ta | Chaqimlar, muammolar |
